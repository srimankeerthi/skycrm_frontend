# SkyCRM

SkyCRM is a modern Customer Relationship Management (CRM) application that helps businesses manage customers, leads, sales activities, and follow-ups efficiently through a simple and user-friendly interface.

---

## Features

- User Authentication (Login & Registration)
- Secure JWT Authentication
- Customer Management
- Lead Management
- Sales Pipeline Tracking
- Task and Follow-up Management
- Search and Filter Customers
- Dashboard with Business Insights
- Activity Logs
- Responsive User Interface

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/SkyCRM.git
```

Navigate to the project directory:

```bash
cd SkyCRM
```

Install backend dependencies:

```bash
cd server
npm install
```

Install frontend dependencies:

```bash
cd client
npm install
```

---

## Running the Application

Start the backend server:

```bash
npm run dev
```

Start the frontend application:

```bash
npm start
```

---

## Environment Variables

Create a `.env` file inside the `server` directory and add the following:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
